# Huawei Cloud Invoice Converter

This repository is now a Next.js + React + TypeScript application for guiding AWS-to-Huawei Cloud invoice conversions.

## Getting started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Scripts

- `npm run dev` – start the dev server
- `npm run build` – create a production build
- `npm run start` – run the production server
- `npm run lint` – run linting checks

## Docker

```bash
docker build -t huawei-cloud-invoice-conversion .
docker run -p 3000:3000 huawei-cloud-invoice-conversion
```
